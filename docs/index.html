My Research Experience

  • OSS Volunteer Engagement

    Spring 2024 - For a semester, I worked with the UMass Public Policy department to explore & visualize the relationship between volunteer sabbaticals & success in the Apache incubator program for open source software projects. This project was a continuation of previous studies looking at different factors affecting volunteer retention in OSS incubating projects. We used commit records as our representation of activity & the Bayesian change-point R package to highlight significant changes in that work activity. Based on our visualizations & statistical tests, this definition of sabbaticals's presence is not strongly correlated with success in the apache incubator program. Further exploration will include using email records to define which dips & returns in productivity were intentional sabbaticals & which were undefined breaks to decrease the noise in our data.

  • Forward Mode Automatic Differentiation

    Summer 2024 - As a part of the UMass CICS URV program, I had the opportunity to work with Kunjal Panchal (PhD canidate) & a team to research, profile, & appraise the new PyTorch Forward Mode Automatic Differentiation Beta version. We used Google Collab's GPUs to train the regular-sized BERT model using the new Beta version of the FW AD, the old Alpha version of the FW AD, and the standard reverse mode ad so that the changes could be compared for their effectiveness. We profiled the memory usage of each auto diff method, tracked the accuracy, & determined the convergence time. Our conclusions were that there were significant improvements in both accuracy & memory usage between the alpha & the beta versions, but both models still perform worse than the backprop option. Further testing could include using a smaller model with less parameters for testing.
